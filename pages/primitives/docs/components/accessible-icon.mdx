description: Useful for wrapping SVG's in an accessible wrapper.
features:
  - Ensures that SVG icons are properly labelled whether or not those labels are visible.
  - Allows screen reader users to more easily understand the context of visual content.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-accessible-icon
```

Then import the component in your component file.

```jsx
import * as AccessibleIconPrimitive from '@radix-ui/react-accessible-icon';
```

### Basic example

```jsx live manual
// CloseIcon.tsx
// import * as AccessibleIconPrimitive from '@radix-ui/react-accessible-icon';
// import { CrossCircledIcon } from '@modulz/radix-icons';

render(
  <AccessibleIconPrimitive.Root label="Close">
    <CrossCircledIcon />
  </AccessibleIconPrimitive.Root>
);
```
