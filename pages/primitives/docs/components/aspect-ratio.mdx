### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-aspect-ratio
```

Then import the component in your component file.

```jsx
// Avatar.tsx
import * as AspectRatioPrimitive from '@interop-ui/react-aspect-ratio';
```

### Basic example

```jsx live manual
// AspectRatio.tsx
// import * as AspectRatioPrimitive from '@interop-ui/react-aspect-ratio';

render(
  <AspectRatioPrimitive>
    <img src="https://picsum.photos/id/1006/800/800" alt="mountain view" />
  </AspectRatioPrimitive>
);
```
