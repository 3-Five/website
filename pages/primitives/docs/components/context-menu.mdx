---
title: Context Menu
name: context-menu
version: 0.0.1-1
description: Displays a menu located at the pointer, triggered by a right-click.
features:
  - Supports items, labels, groups of items.
  - Supports checkable items (single or multiple).
  - Customise side, alignment, offsets, collision handling.
  - Focus is fully managed.
  - Full keyboard navigation.
  - Dismissing and layering behavior is highly customisable.
---

### Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-context-menu
```

Then import the component in your component file.

```jsx
// ContextMenu.tsx
import * as ContextMenuPrimitive from '@radix-ui/react-context-menu';
```

### Basic example

```jsx live manual
// ContextMenu.tsx
// import * as ContextMenuPrimitive from '@radix-ui/react-context-menu';

const MenuContent = styled(ContextMenuPrimitive.ContextMenuContent, {
  minWidth: 130,
  backgroundColor: 'white',
  borderRadius: 6,
  padding: 5,
  boxShadow:
    'hsla(206,22%,7%,.15) 0px 5px 15px -5px, hsla(206,22%,7%,.1) 0px 5px 10px -15px',
});

const MenuItem = styled(ContextMenuPrimitive.ContextMenuItem, {
  fontSize: 13,
  padding: '5px 10px',
  borderRadius: 3,
  cursor: 'default',

  '&[data-disabled]': { color: 'slategray' },

  ':focus': {
    outline: 'none',
    backgroundColor: 'dodgerblue',
    color: 'white',
  },
});

const MenuSeparator = styled(ContextMenuPrimitive.ContextMenuSeparator, {
  height: 1,
  backgroundColor: 'gainsboro',
  margin: '5px 10px',
});

const TriggerArea = styled(ContextMenuPrimitive.ContextMenuTrigger, {
  display: 'block',
  textAlign: 'center',
  padding: 100,
  backgroundColor: 'slategray',
});

render(
  <ContextMenuPrimitive.Root>
    <TriggerArea>Right click anywhere here</TriggerArea>
    <MenuContent sideOffset={-5}>
      <MenuItem onSelect={() => alert('undo')}>Undo</MenuItem>
      <MenuItem onSelect={() => alert('redo')}>Redo</MenuItem>
      <MenuSeparator />
      <MenuItem disabled onSelect={() => alert('cut')}>
        Cut
      </MenuItem>
      <MenuItem onSelect={() => alert('copy')}>Copy</MenuItem>
      <MenuItem onSelect={() => alert('paste')}>Paste</MenuItem>
    </MenuContent>
  </ContextMenuPrimitive.Root>
);
```

### API Reference

#### ContextMenu

`ContextMenu` is the root component.

#### Trigger

`ContextMenuTrigger` is the area that triggers the `ContextMenu`. Wrap this around the thing you want your `ContextMenu` to open when right-clicking (or using the relevant keyboard shortcuts).

#### Content

`ContextMenuContent` is the component that pops out in an open `ContextMenu`.
