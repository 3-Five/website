## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-toggle-button
```

## Anatomy

Import the component.

```jsx
import { ToggleButton } from '@radix-ui/react-toggle-buton';

export default () => <ToggleButton />;
```

## Basic Example

Create your styled toggle button component from the primitive parts.

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](../overview/styling).

```jsx live manual
// import { styled } from 'path-to/stitches.config';
import * as ToggleButtonPrimitive from '@radix-ui/react-toggle-button';

const ToggleButton = styled(ToggleButtonPrimitive.Root, {
  appearance: 'none',
  backgroundColor: 'transparent',
  border: 'none',
  padding: '5px 10px',
  boxShadow: 'inset 0 0 0 1px gainsboro',
  overflow: 'hidden',
  borderRadius: 3,

  '&:focus': {
    outline: 'none',
    boxShadow: 'inset 0 0 0 1px dodgerblue, 0 0 0 1px dodgerblue',
  },

  '&[data-state=on]': {
    backgroundColor: 'gainsboro',
  },
});

export default () => (
  <>
    <ToggleButton>Togggle</ToggleButton>
  </>
);
```
